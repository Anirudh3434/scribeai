Privacy Policy

Last updated: June 20, 2026

info 1. Introduction

Welcome to ScribeAI. We are committed to protecting your privacy. ScribeAI is a premium, privacy-first offline AI writing assistant for macOS. This policy describes how we collect, use, and process your data.

offline_bolt 2. Local Processing (Our Core Promise)

ScribeAI operates 100% locally on your computer when using the default offline model (WebGPU execution). All text refinement, grammar checks, rewrites, and AI processing occur entirely on your local CPU/GPU. No text logs, inputs, or drafts are ever transmitted to our servers or any third-party servers. Your work remains completely private on your hardware.

cloud 3. Optional Cloud API Keys

If you explicitly configure and select an optional Cloud API (such as Gemini Cloud API) and provide your own API key, ScribeAI will communicate directly with that provider's official API endpoint.

Your API key and prompt text are sent directly to the selected API provider and are never routed through ScribeAI or any intermediate proxy server. Your API keys are stored securely in your macOS local Keychain or local storage.

analytics 4. Website Analytics & Cookie Preferences

Our website (scribeai.shop) uses Google Analytics 4 (GA4) to track anonymous aggregates (such as clicks on download links or website visits) to help us optimize the landing page experience. No personally identifiable information or app text data is tracked.

We use the Secure Privacy consent tool to respect your cookie consent preferences and comply with GDPR, CCPA, and privacy laws. You can adjust your consent settings at any time using the banner widget on the site.

vpn_key 5. Data Security

Since all data is stored and processed locally, the security of your data depends on the security of your local device. We do not maintain any databases of user text, user credentials, or AI prompt histories.

mail 6. Contact & Your Rights

Since we do not collect or store your personal data, we have no records to access, delete, or modify. If you have questions about this policy, you can contact us via our GitHub Repository issue tracker.
